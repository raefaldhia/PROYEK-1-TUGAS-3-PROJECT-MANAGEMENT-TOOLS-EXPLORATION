# DATE

Mengembalikan format berupa tanggal dari argumen yang telah diberikan.

#### Syntax

```text
=DATE(year;month;day)
```

| Parameter | Deskripsi |
| :--- | :--- |
| year | Angka dari tahun. |
| month | Angka dari bulan. |
| day | Angka dari hari. |

## Contoh

| A | Hasil |
| :--- | :--- |
| =DATE\(2019;3;9\) | 09/03/2019 |
