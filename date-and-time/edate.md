# EDATE

Mengembalikan nomor serial merepresentasikan tanggal yang berindikator banyaknya bulan sebelum atau sesudah start\_date.

#### Syntax

```text
=EDATE(start_date;month)
```

| Parameter | Deskripsi |
| :--- | :--- |
| start\_date | Tanggal yang merepresentasikan tanggal awal |
| month |  |
