# TYPE

Mengembalikan nilai berupa integer sebagai representasi dari tipe data value yang diberikan. \(1\) Number, \(2\) Text, \(4\) Logical value\), \(16\) Error value, \(64\) Array.

## Syntax

```text
=TYPE(value)
```

| Parameter | Deskripsi |
| :--- | :--- |
| value | Hanya dapat diisi oleh Microsoft Excel value. |
