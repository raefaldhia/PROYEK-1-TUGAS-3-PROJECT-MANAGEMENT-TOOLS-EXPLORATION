# DATE

#### DATE

Mengembalikan format berupa tanggal dari argumen yang telah diberikan.

```text
=DATE(year;month;day)
```

| Parameter | Deskripsi |
| :--- | :--- |
| year | tahun |
| month | bulan |
| day | hari |

{% tabs %}
{% tab title="Contoh 1" %}
| A |  |
| :--- | :--- |
| =DATE\(2019;3;9\) |  |

Result:

| A |  |
| :--- | :--- |
| 09/03/2019 |  |
