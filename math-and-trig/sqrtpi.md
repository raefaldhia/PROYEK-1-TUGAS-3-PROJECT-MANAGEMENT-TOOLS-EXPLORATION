# SQRTPI

### Syntax

{% code-tabs %}
{% code-tabs-item title="Sintaks Fungsi" %}
```text
SQRTPI ( Number )
```
{% endcode-tabs-item %}
{% endcode-tabs %}

### Parameter

| Parameter | Deskripsi |
| :--- | :--- |
| Number  | Nilai numerik positif, yang ingin Anda kalikan dengan pi, sebelum menghitung akar kuadrat. |

{% hint style="info" %}


> #### Jika angka &lt; 0, SQRTPI mengembalikan nilai kesalahan \#NUM!.
{% endhint %}
